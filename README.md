# Boilerplate Next.js v2

## Requirments

-   NodeJs 14.15+
-   NPM 6.14+ `or` yarn 1.22+

## Technology

-   Next.js 10.0.6+
-   TypeScript 4.0.3+
-   ESLint 7.12.1+
-   Redux 4.0.5+

## How to use

-   Install NodeJS 14+
-   Clone the Repo
-   Run `npm install` or `yarn`
-   Run `yarn dev`

Contact [@sonjoydatta](https://github.com/sonjoydatta) for and example `.env` file for development and production.
